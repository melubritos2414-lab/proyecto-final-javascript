 README — Mirna Web Studio
Un proyecto realizado con dedicación, aprendizaje y mucha práctica
 Sobre el proyecto
Este proyecto forma parte del curso de desarrollo web y representa mi primera web completa con varias páginas, navegación, estilos propios y un mapa dinámico integrado.
Mi objetivo fue crear un sitio sencillo, funcional y visualmente agradable, aplicando todo lo aprendido en HTML, CSS y JavaScript.
 Qué incluye la web
• 	Página de inicio con accesos rápidos
• 	Galería de imágenes
• 	Formulario de presupuesto
• 	Página de contacto con mapa dinámico y ruta por carretera
• 	Footer unificado en todas las páginas
• 	Carga de noticias desde un archivo JSON
• 	Diseño oscuro y moderno
 Mapa dinámico
La página de contacto incorpora un mapa interactivo con Leaflet y Leaflet Routing Machine.
El mapa muestra:
• 	La ubicación del negocio
• 	La ubicación del usuario (si da permiso)
• 	Una ruta real por carretera, tal como se pedía en el ejercicio
• 	Instrucciones paso a paso del recorrido
Este fue uno de los apartados más desafiantes, pero también uno de los que más disfruté.
 Tecnologías utilizadas
• 	HTML5
• 	CSS3
• 	JavaScript
• 	Leaflet + Leaflet Routing Machine
• 	JSON para las noticias
• 	Font Awesome
